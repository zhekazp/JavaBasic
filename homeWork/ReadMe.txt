The repository contains solutions to tasks provided in the Java language learning course.

The classes folder contains classes that are used by almost all applications presented in this repository.
The UserInput Class is used to enter data from the keyboard.

The homeWorks folder contains solutions to tasks posed by the teacher after studying certain material.

The extraHomeWorks folder contains solutions to tasks that were provided to reinforce the knowledge.

The Projects folder contains the final projects posed by the teacher after completion of the topics.
